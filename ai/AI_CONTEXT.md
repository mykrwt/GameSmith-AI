# AI Context

Welcome to the AI 3D Game Builder repository. This directory (`/ai`) acts as the source of truth for all AI agents working on this project. 

## Purpose of this Folder
This folder contains foundational instructions, rules, and structural guidelines to ensure that AI-generated code does not break the project over time. As an AI agent, you must read and adhere to the guidelines in these documents before making any structural changes to the repository.

## Files Explaination
- **[GOAL.md](./GOAL.md)**: Defines the core objective of the AI 3D Game Builder project. It explains what we are trying to build.
- **[RULES.md](./RULES.md)**: Contains critical constraints for AI agents to ensure modular code, safeguard core systems, and enforce long-term architecture stability.
- **[ARCHITECTURE.md](./ARCHITECTURE.md)**: Details the system's high-level architectural layers (`ENGINE`, `GENERATOR`, `TEMPLATES`, `EDITOR`, `RUNTIME`, and `GAMES`) and their responsibilities.
- **[CODEMAP.md](./CODEMAP.md)**: Explains the repository's folder structure and acts as a map to denote where different types of functionalities belong.
